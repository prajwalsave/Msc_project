# Analysis Folder

This directory contains scripts and notebooks used for data preprocessing and analysis.

## Structure
- `data/`: Raw input data (gitignored).
- `outputs/`: Generated plots (PNG) and processed data (CSV, JSON).
- `fetch_extended_aq.py`: Aggregates historical OpenAQ data.
- `build_traffic_profile.py`: Generates the 7x24 traffic profile JSON.
- `01_g1_corr.ipynb`: Correlation analysis between traffic and AQ.
